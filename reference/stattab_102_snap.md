# Snapshot of STAT-TAB px 102

Used for tests and to make it easier for users to look up the spelling
used on STAT-TAB.

## Usage

``` r
stattab_102_snap
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 177
rows and 6 columns.

## Source

Data obtained from the Swiss Federal Statistical Office (FSO):

- <https://www.pxweb.bfs.admin.ch/pxweb/en/px-x-0104020000_102/-/px-x-0104020000_102.px>
