# Snapshot of STAT-TAB that includes the population records

Used for tests and to make it easier for users to look up the spelling
used on STAT-TAB.

## Usage

``` r
stattab_pop_snap
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with
2431 rows and 7 columns.
