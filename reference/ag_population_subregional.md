# Population data for the canton of Aargau (five subregions)

Population from the canton of Aargau divided into five subregions.

## Usage

``` r
ag_population_subregional
```

## Format

The example population records include the number of people of each
demographic group (nationality (2) X sex (2) X age classes (101)) for
five spatial units of the canton of Aargau in 2025.
